# Just Cos

Just Cos is a mobile-first trig learning calculator. It solves triangles from user-entered sides and angles, draws the resulting triangle, shows the formula path, and explains missing or impossible data in plain language.

## Run locally

Open `index.html` in a browser.

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the source to the `main` branch and the root folder.
4. Save, then open the Pages URL GitHub provides.
